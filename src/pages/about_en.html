What I build for myself


My personal projects go low-level and long-term. The main one is Ex-Nihilo Core: a RV32 RISC-V emulator written in C. I built a real boot flow that can run a Linux kernel (OpenSBI + DTB + kernel image). It’s the kind of project that forces you to respect specs, debugging, and “truth over vibes”.


I also mess around with hardware (FPGA/microcontrollers/Raspberry Pi/ESP32) and I’m slowly moving toward a bigger goal: build my own RISC-V CPU and run Linux on it.
